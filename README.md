# Test News Application

A Laravel-based news platform with role-based access control and real-time notifications.

## About This Project

This is a news management application built with Laravel that provides a complete content management system with user authentication, role-based permissions, and real-time notifications. The platform allows users to create, manage, and consume news content with instant updates for content authors.

## Key Features

- **User Authentication** - Complete authentication system with registration, login, and logout functionality
- **Role-Based Access Control** - Users are assigned specific roles with different permission levels (e.g., Admin, Author, Reader)
- **News Management** - Create, read, update, and delete news articles
- **Real-Time Notifications** - WebSocket-based notifications to alert authors about actions related to their content
- **User Management** - Comprehensive user management system with role assignment

## Technology Stack

- **Backend Framework**: Laravel
- **Real-Time Communication**: WebSockets for instant notifications
- **Authentication**: Laravel built-in authentication
- **Authorization**: Role-based access control system
